parallax

Vi ville gerne have en form for interactive baggrund da vi fandt parallax.
Vi har leget en del med det og det er blevet acceptabelt men det er noget som vi vil arbjde mere på.

HTML Code:

<div class="parallax"> <div id="group1" class="parallax__group"> <div class="parallax__layer--base"></div> <div class="parallax__layer parallax__layer--back"></div> <div class="parallax__layer parallax__layer--deep"></div> </div> </div>

CSS Code:

.parallax { font-size: 200%; perspective: 1px; height: 100vh; overflow-x: hidden; overflow-y: auto; text-align: center; } .parallax__layer { padding: 96vh 0; position: absolute; top: 0; right: 0; bottom: 0; left: 0; } .parallax__layer--base { transform: translateZ(0); } .parallax__layer--back { transform: translateZ(-1px); left: -55%; top: -110%; } .parallax__layer--back img{ width: 135%; position: relative; height: auto; align-items: center; } .parallax__layer--deep { transform: translateZ(-2px) scale(3); position: absolute; left: -1%; width: 101%; background-image: url(../Pictures/Background/BackGround-LayerDeep.png); } .parallax__group { position: relative; height: 125vh; transform-style: preserve-3d; }

Video Dropdown

Vi ville gerne have at man kunne have lidt mere interaktivtet med vores hjemmeside så vi lavede en Video-Dropdown.
og det er blevet mere end acceptabelt men det er stadigvæk begrænste hvad man kan lave uden et java script

HTML Code:

<div class="dropdown pics"> <img src="./pictures/musikgalPics/arashi-love1.jpg" alt="Arnol in japan" width="200" height="125"> <div class="dropdown-content"> <iframe width="420" height="315" src="https://www.youtube.com/embed/SE8Syp3S8tQ"></iframe> <div class="desc">Arashi - Hold It Up</div> </div> </div>

CSS Code:

.dropdown { position: relative; display: inline-block; float: right; } .dropdown-content { display: none; position: absolute; left: 0; background-color: #f9f9f9; min-width: 160px; box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2); z-index: 1; } .dropdown:hover .dropdown-content { display: block; }

keyframes

Vi ville gerne prøve at lege lidt med keyframes på vores hjemmeside så vi lavede nogen flotte animationer.
og selv om at det ser ngoet sjovt ud lærte vi en god del om det

HTML Code:

<img class="cusPics2 borderclass anime" src="./pictures/ArnolInJapan.jpg" alt="Arnol in japan"> <img class="cusPics3 borderclass anime" src="./pictures/Bobanana.jpg" alt="Bob with Bobanana"> <img class="cusPics1 borderclass anime" src="./pictures/mariojapan.jpg" alt="">

CSS Code:

.anime { animation: mymove 3s infinite; animation-direction: alternate; animation-timing-function: linear; } @keyframes mymove { from{ left: 50%; opacity: 1; transform: rotate(-25deg); } to{ left: 50%; opacity: 0.1; transform: rotate(125deg); } }
